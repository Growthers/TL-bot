# TL-Bot  

timesカテゴリーの投稿を一つのTLっぽく表示するDiscordBotです  

## Run  

`Linux, macOS`  
```shell
$ python3 -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
$ python3 main.py
```

`Windows`  
```shell
$ python3 -m venv venv
$ .\venv\Scripts\activate
$ pip install -r requirements.txt
$ python3 main.py
```
